📝 SX_WarehouseCraft

🧠 About
This is a simple warehouse crafting script that i created for fun on the ESX Legacy Framework and decided to share as my first repository. It comes with a custom warehouse mlo if you do not already have one. It has an import.sql for adding crafting logs into your data base so you can see exactly who crafted what and at what time. My favorite part about it is the crafting history tab where a player can see their last 10 crafts. But you'll see for yourself, everything is listed below including things you need and install instructions to get this resource loaded up into your ESX server with zero problems.

⚙️ What You Need

oxmysql

ox_lib

ox-inventory

ox_target

🖥️ Install

1: Open the folder and drag the entire "warehouse mlo" folder to your server resources and ensure it in your server.cfg.
2: Take the entire wh_craft folder and drag it into your server resources and make sure you ensure it after the "warehouse mlo".

ensure warehouse mlo
ensure wh_craft

3: Open import.sql inside the "wh_craft" resource and paste the code inside there into a database query to create the crafting logs table.
4: Look at the config inside the shared folder and make sure everything is tuned to your liking.
5: Dont worry about the images folder, just boot up your server and enjoy.
